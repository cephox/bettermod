# BetterMod
A better discord moderation bot

## Usage
### Requirements
- python >=3.8:
  - discord.py
  - pymongo
- mongodb

### Initialisation
```bash
git clone https://github.com/cephox/bettermod
cd bettermod

python bettermod.py
```

### Reset the bot and database
```bash
python bettermod.py reset
```
